# Vehicle Sales Dashboard

A JavaFX desktop application that visualizes vehicle sales data through interactive charts and tables. Built as part of university coursework at Liverpool John Moores University.

## Features

- **Interactive Charts** - Bar, Pie, and Line charts for visualizing sales data
- **Data Table** - Browse, filter, and analyze sales records with multi-row selection
- **REST API Integration** - Fetches real-time data from a remote server
- **Offline Support** - Automatic JSON backup when network is unavailable
- **Dynamic Filtering** - Filter by year and vehicle type

## Technologies Used

- Java 8
- JavaFX & FXML
- Gson (JSON parsing)
- CSS
- REST API

- Object-Oriented Programming
